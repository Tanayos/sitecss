La route vers l'illumination

Les anciennes reliques de balises spécifiques au navigateur, les DOM incompatibles , le support CSS brisé et les navigateurs désaffectés sont restés sur une route sombre et morne .

Nous devons effacer l'esprit du passé. L'illumination du Web a été réalisée grâce aux efforts inlassables de gens comme le W3C , WaSP et les principaux créateurs de navigateurs.

Le CSS Zen Garden vous invite à vous détendre et à méditer sur les leçons importantes des maîtres. Commencer à voir avec clarté. Apprenez à utiliser les techniques ancestrales de manière nouvelle et revigorante. Devenez un avec le web.

Alors, de quoi s'agit-il?

Il y a un besoin continu de montrer la puissance de CSS . Le Jardin Zen a pour but d'exciter, d'inspirer et d'encourager la participation. Pour commencer, affichez certains des modèles existants dans la liste. En cliquant sur n'importe quel chargera la feuille de style dans cette même page. Le HTML reste le même, la seule chose qui a changé est le fichier CSS externe . Oui vraiment

CSS permet un contrôle complet et total sur le style d'un document hypertexte. La seule façon d'illustrer cela de manière à susciter l'enthousiasme des gens est de montrer ce que cela peut vraiment être, une fois que les rênes sont entre les mains de ceux qui sont capables de créer la beauté à partir de la structure. Les concepteurs et les codeurs ont contribué à la beauté du Web; nous pouvons toujours pousser plus loin.

Participation

Un design visuel fort a toujours été notre objectif. Vous êtes en train de modifier cette page, de sorte que de solides compétences en CSS sont également nécessaires, mais les fichiers d'exemple sont suffisamment commentés pour que même les novices CSS puissent les utiliser comme points de départ. Veuillez consulter le guide de ressources CSS pour des tutoriels avancés et des conseils sur l'utilisation de CSS .

Vous pouvez modifier la feuille de style comme vous le souhaitez, mais pas le HTML . Cela peut sembler décourageant au début si vous n'avez jamais travaillé de cette façon auparavant, mais suivez les liens listés pour en savoir plus et utilisez les exemples de fichiers comme guide.

Avantages

Pourquoi participer? Pour la reconnaissance, l'inspiration et une ressource, nous pouvons tous faire référence à montrer aux gens à quel point CSS peut vraiment être incroyable . Ce site sert d'inspiration à tous ceux qui travaillent sur le web aujourd'hui, d'outil d'apprentissage pour ceux qui seront demain et de galerie de techniques futures que nous pouvons tous espérer.

Exigences

Dans la mesure du possible, nous aimerions voir principalement l'utilisation de CSS 1 & 2 . CSS 3 & 4 devrait être limité aux éléments largement supportés seulement, ou de forts replis devraient être fournis. Le CSS Zen Garden est consacré aux CSS fonctionnels et pratiques et non aux dernières astuces à la mode visibles par 2% des internautes. La seule exigence réelle que nous avons est que votre CSS valide.

Heureusement, la conception de cette façon montre à quel point les différents navigateurs ont implémenté CSS maintenant. Lorsque vous respectez les consignes, vous devriez voir des résultats assez cohérents dans la plupart des navigateurs modernes. En raison du grand nombre d'agents utilisateurs sur le Web ces jours-ci - en particulier lorsque vous prenez en compte le mobile - les mises en page au pixel près ne sont pas toujours possibles sur toutes les plates-formes. C'est bon, mais faites le test autant que vous le pouvez. Votre design devrait fonctionner dans au moins IE9 + et les derniers navigateurs Chrome, Firefox, iOS et Android (gérés par plus de 90% de la population).

Nous vous demandons de soumettre une œuvre d'art originale. Veuillez respecter les lois sur les droits d'auteur. S'il vous plaît gardez le matériel répréhensible au minimum, et essayez d'incorporer des thèmes visuels uniques et intéressants à votre travail. Nous avons dépassé le point d'avoir besoin d'un autre design lié au jardin.

C'est un exercice d'apprentissage ainsi qu'une démonstration. Vous conservez tous les droits d'auteur sur vos graphiques (à quelques exceptions près, voir les directives de soumission ), mais nous vous demandons de libérer votre CSS sous une licence Creative Commons identique à celle de ce site afin que d'autres puissent apprendre de votre travail.
